# Отчёт по классификации самолётов

##  Результаты
- **ViT Accuracy**: 0.9667 (≈ 96.7%)  
  → правильно классифицировано 29 изображений из 30
- **ResNet Accuracy**: 0.9000 (≈ 90.0%)  
  → правильно классифицировано 27 изображений из 30
##  Цель
Классификация изображений самолётов на две категории:
- **Гражданские (Civil)**
- **Военные (Military)**

Использованы два предобученных модели из Hugging Face:
- Vision Transformer (ViT) – `google/vit-base-patch16-224`
- ResNet-50 – `microsoft/resnet-50`

##  Датасет
- Количество изображений: 30 фотографий самолётов
- Классы: civil, military
- Имена файлов отражают истинную метку (например: `civil_01.jpg`, `military_12.png`)
- В выборке присутствуют нетривиальные случаи (сложные ракурсы, шумный фон, удалённые самолёты)

##  Методология
1. **Предобработка**: изображения преобразованы в RGB и обработаны встроенными процессорами моделей.
2. **Предсказание**: модели выдают вероятности для 1000 классов ImageNet → выполнено сопоставление с классами civil/military.
3. **Оценка**: рассчитана точность (Accuracy) для каждой модели.

##  Наблюдения
- Модель ViT показала лучшие результаты по сравнению с ResNet (разница около 6.7%).
- Обе модели испытывали трудности с изображениями, где самолёты были маленькими или частично закрытыми.
- Гражданские самолёты классифицировались точнее, чем военные.

## Вывод
Предобученные модели Hugging Face можно адаптировать для бинарной классификации самолётов с высокой точностью.  
ViT показал лучший результат в данном эксперименте. Для повышения точности рекомендуется выполнить **fine-tuning** на более крупном и разнообразном датасете.
